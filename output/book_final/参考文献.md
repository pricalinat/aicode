# 参考文献

本书内容基于公开学术论文、行业实践以及本地论文知识库的系统性分析整理而成。本书利用了包含423篇论文的本地知识库，按章节分类整理，每章引用的主要论文来源见各章末尾。

## 本地论文知识库

本书参考了经过系统性整理的本地论文知识库，该知识库包含以下章节的论文资源：

| 章节 | 论文数量 | 主题领域 |
|------|---------|----------|
| 第1章 - 基础概念 | 45篇 | 嵌入表示、预训练模型 |
| 第2章 - 深度学习 | 22篇 | Transformer、BERT、GPT |
| 第3章 - 知识图谱 | 27篇 | 知识表示、图神经网络 |
| 第4章 - 商品理解 | 275篇 | 商品搜索、推荐系统 |
| 第5章 - 服务理解 | 26篇 | 对话系统、意图识别 |
| 第6章 - 双向匹配 | 5篇 | 搜推一体化 |
| 第7-10章 - 评测体系 | 23篇 | 评测方法、指标、LLM-as-Judge |
