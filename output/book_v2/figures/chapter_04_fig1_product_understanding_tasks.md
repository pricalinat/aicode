│  │  • 小样本分类 (元学习)                                     │   │

│  │  • 排序学习 (LTR)                                         │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                              ↓                                        │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │                    商品匹配                                  │   │
│  │  • 文本匹配 (表示型/交互型)                                 │   │
│  │  • 图像匹配 (Siamese网络)                                  │   │
│  │  • 多模态融合匹配                                          │   │
│  └──────────────────────────────────────────────────────────────┘   │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
