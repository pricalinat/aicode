# 图S.1：全书知识体系架构图

## 整体架构设计

本书的知识体系分为五个层次，从基础理论到实践应用：

```mermaid
flowchart TD
    subgraph 第五层：应用实践
        A[电商搜索] --> A1[推荐系统]
        A --> A2[智能客服]
        A --> A3[内容生成]
    end
    
    subgraph 第四层：服务运营
        B[服务评测] --> B1[效果评估]
        B --> B2[性能监控]
        B --> B3[安全合规]
        C[数据闭环] --> C1[数据采集]
        C --> C2[模型迭代]
        C --> C3[持续优化]
    end
    
    subgraph 第三层：核心能力
        D[需求理解] --> D1[意图识别]
        D --> D2[语义编码]
        D --> D3[上下文理解]
        E[供给理解] --> E1[实体识别]
        E --> E2[属性抽取]
        E --> E3[知识融合]
        F[匹配排序] --> F1[向量检索]
        F --> F2[排序模型]
        F --> F3[重排优化]
    end
    
    subgraph 第二层：技术支撑
        G[知识图谱] --> G1[图谱构建]
        G --> G2[图谱嵌入]
        G --> G3[图谱应用]
        H[预训练模型] --> H1[模型结构]
        H --> H2[训练策略]
        H --> H3[模型压缩]
    end
    
    subgraph 第一层：理论基础
        I[深度学习] --> I1[神经网络]
        I --> I2[优化方法]
        J[NLP基础] --> J1[词向量]
        J --> J2[序列模型]
    end
    
    I --> G
    J --> G
    G --> D
    G --> E
    H --> D
    H --> E
    H --> F
    D --> B
    E --> B
    F --> B
    B --> A
    C --> B
```
