# 图3.5：知识图谱嵌入模型对比

```
┌─────────────────────────────────────────────────────────────────────┐
│                    知识图谱嵌入模型对比                              │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│ TransE: h + r = t                                                   │
│ ┌─────────────────────────────────────────────────────────────┐     │
│ │  优点: 简单高效，适合1-1关系                                  │     │
│ │  缺点: 难以处理1-N、N-1、N-N关系                            │     │
│ │  示例: 中国 + 首都 ≈ 北京                                    │     │
│ └─────────────────────────────────────────────────────────────┘     │
│                                                                      │
│ TransH: h⊥ + r = t⊥ (投影到超平面)                                 │
│ ┌─────────────────────────────────────────────────────────────┐     │
│ │  改进: 实体在不同关系下有不同表示                            │     │
│ │  效果: 更好地处理复杂关系                                   │     │
│ └─────────────────────────────────────────────────────────────┘     │
│                                                                      │
│ TransR: Mr × h + r = Mr × t (关系特定变换)                         │
│ ┌─────────────────────────────────────────────────────────────┐     │
│ │  改进: 实体和关系在不同空间                                 │     │
│ │  缺点: 参数量大                                             │     │
│ └─────────────────────────────────────────────────────────────┘     │
│                                                                      │
│ DistMult: h ◦ r ◦ t (双线性)                                       │
│ ┌─────────────────────────────────────────────────────────────┐     │
│ │  改进: 使用对角矩阵简化计算                                 │     │
│ │  限制: 只能建模对称关系                                     │     │
│ └─────────────────────────────────────────────────────────────┘     │
│                                                                      │
│ ComplEx: 复数向量表示                                               │
│ ┌─────────────────────────────────────────────────────────────┐     │
│ │  改进: 使用复数，可以建模非对称关系                          │     │
│ │  示例: Re(h ⊙ r ⊙ t̄)                                       │     │
│ └─────────────────────────────────────────────────────────────┘     │
│                                                                      │
│ ConvE: 2D卷积                                                       │
│ ┌─────────────────────────────────────────────────────────────┐     │
│ │  改进: 使用卷积操作捕获实体交互                              │     │
│ │  结构: h和r reshape为2D后进行卷积                          │     │
│ └─────────────────────────────────────────────────────────────┘     │
│                                                                      │
│ RotatE: h ⊙ r = t (旋转)                                           │
│ ┌─────────────────────────────────────────────────────────────┐     │
│ │  改进: 关系建模为旋转，可建模多种关系模式                    │     │
│ │  支持: 对称/反对称/反转/组合关系                            │     │
│ └─────────────────────────────────────────────────────────────┘     │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```
