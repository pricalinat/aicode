# 图1.5：多模态商品理解技术

## CLIP多模态对比学习框架

```
图像编码器 (ViT):
┌──────────────────────────────────────────────┐
│  Image Patch 1 → Linear + Pos Embed         │
│  Image Patch 2 → Linear + Pos Embed         │
│  ...                                        │
│  Image Patch N → Linear + Pos Embed         │
│                    ↓                         │
│         Transformer Encoder                  │
│                    ↓                         │
│              Image Embedding I               │
└──────────────────────────────────────────────┘

文本编码器 (BERT):
┌──────────────────────────────────────────────┐
│  Token Embedding + Position Embedding       │
│                    ↓                         │
│         Transformer Encoder                  │
│                    ↓                         │
│              Text Embedding T                │
└──────────────────────────────────────────────┘

对比学习目标:
┌──────────────────────────────────────────────┐
│  Sim(I, T) = cos(I, T)                      │
│  Loss = -1/2 [log(exp(sim(I_i,T_i))/Σexp)  │
│                + log(exp(sim(T_i,I_i))/Σexp)]│
└──────────────────────────────────────────────┘
```
