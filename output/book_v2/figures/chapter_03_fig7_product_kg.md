# 图3.7：商品知识图谱数据模型

```
┌─────────────────────────────────────────────────────────────────────┐
│                      商品知识图谱数据模型                           │
├─────────────────────────────────────────────────────────────────────┤
│                                                                      │
│  实体类型 (Entity Types):                                           │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  Product (商品)      - 具体商品SKU                           │   │
│  │  Category (类目)     - 商品分类层次                          │   │
│  │  Brand (品牌)        - 品牌信息                              │   │
│  │  Attribute (属性)    - 属性定义                             │   │
│  │  Shop (店铺)         - 卖家店铺                              │   │
│  │  User (用户)         - 消费者                                │   │
│  │  Review (评价)       - 用户评价                              │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                      │
│  关系类型 (Relation Types):                                          │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  has_brand                 所属品牌                          │   │
│  │  belongs_to_category       所属类目                          │   │
│  │  has_attribute             含有的属性                        │   │
│  │  similar_to                相似商品                          │   │
│  │  complementary_to         互补商品                           │   │
│  │  also_buy                 经常一起购买                       │   │
│  │  is_a                     是一种(品类)                      │   │
│  │  sold_by                  卖家                               │   │
│  │  reviewed_by              评价                              │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                      │
│  示例:                                                               │
│  ┌─────────────────────────────────────────────────────────────┐   │
│  │  (iPhone15, has_brand, 苹果)                               │   │
│  │  (iPhone15, belongs_to_category, 智能手机)                 │   │
│  │  (iPhone15, has_attribute, A15芯片)                        │   │
│  │  (iPhone15, similar_to, 三星S23)                          │   │
│  │  (iPhone15, complementary_to, 手机壳)                     │   │
│  │  (iPhone15, also_buy, AirPods Pro)                        │   │
│  │  (智能手机, is_a, 数码产品)                                 │   │
│  │  (AirPods Pro, also_buy, iPhone15)                        │   │
│  └─────────────────────────────────────────────────────────────┘   │
│                                                                      │
└─────────────────────────────────────────────────────────────────────┘
```
