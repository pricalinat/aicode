## 异常检测模块 (已升级)

### 参考论文
- **LUNAR** (arXiv:2002.09594) - GNN + 可学习邻居聚合
- **OCGNN** (arXiv:2112.05355) - One-Class GNN
- **AnomalyDAE** (arXiv:2308.02914) - 对比学习自编码器

### 检测模式
| 模式 | 方法 | 说明 |
|------|------|------|
| `legacy` | 质心+KNN距离 | 原有方法 |
| `one_class` | One-Class SVM | 找边界球心 |
| `reconstruction` | SVD重构误差 | 特征+邻居重构 |
| `knn_distribution` | KNN分布 | 分布异常 |
| `ensemble` | 加权集成 | 综合以上三种 (默认) |

### 输出指标
- `legacy_score`
- `one_class_score` + radius
- `reconstruction_score` + feature/neighbor error
- `knn_distribution_score`
