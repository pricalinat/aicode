# 书籍图表生成说明

## 概述

本文档包含第5-8章共19张技术图表，每章涵盖不同的技术主题。

## 使用方法

所有图表均为PlantUML格式，导入Draw.io步骤：
1. 打开 Draw.io (draw.io)
2. 点击 Arrange > Insert > Advanced > PlantUML (SVG)
3. 粘贴对应的PlantUML代码
4. 点击 Insert 即可编辑和导出

## 导出格式

导入Draw.io后可导出为：
- PNG (图片)
- SVG (矢量图)
- PDF (文档)
- XML (可编辑)

建议导出PNG或SVG用于书籍排版。
