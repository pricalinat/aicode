# 属性关系分析 Skill

## 目标
分析属性之间的关系，构建属性网络

## 关系类型

### 1. 上下位关系
- 上位：电子产品
- 下位：手机→智能手机→iPhone

### 2. 互斥关系
- 颜色：红色 vs 蓝色（互斥）

### 3. 依赖关系
- 规格依赖品类
- 手机壳依赖手机品类

### 4. 组合关系
- 颜色+尺寸 组合

### 5. 同义关系
- 手机=移动电话=手机终端
