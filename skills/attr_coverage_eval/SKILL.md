# 属性覆盖率评估 Skill

## 目标
评估当前属性库是否覆盖80%场景

## 评估方法

### 1. 属性项覆盖率
- 当前有多少属性项？
- 目标：1000+

### 2. 属性值覆盖率
- 每个属性项有多少属性值？
- 目标：10,000+ 总属性值

### 3. 场景覆盖率
- 这些属性能覆盖多少商品？
- 抽查100个商品验证覆盖率
- 目标：80%

## 决策树

```
IF 覆盖率 >= 80%:
    任务完成 ✅
ELIF 属性项 >= 1000 AND 覆盖率 < 80%:
    继续扩展到5000属性项
ELIF 属性项 >= 5000 AND 覆盖率 < 80%:
    继续扩展到10000属性项
ELSE:
    继续挖掘更多属性
```
