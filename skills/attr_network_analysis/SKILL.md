# 属性关系网络分析 Skill

## 目标
分析属性之间的关系网络

## 关系类型

### 1. 层级关系
- 父属性→子属性
- 例如：品类→二级品类→三级品类

### 2. 互斥关系
- 不能同时存在的属性
- 例如：颜色之间（互斥选择）

### 3. 依赖关系
- 一个属性依赖另一个属性
- 例如：内存依赖手机品类

### 4. 组合关系
- 经常一起出现的属性
- 例如：颜色+尺寸

### 5. 影响关系
- 一个属性影响另一个属性
- 例如：品牌影响价格

## 分析方法

1. 遍历所有属性
2. 识别关系类型
3. 构建关系网络图
4. 找出关键属性（连接最多的属性）
