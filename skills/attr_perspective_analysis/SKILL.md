# 属性视角分析 Skill

## 目标
从不同业务视角分析属性库

## 视角维度

### 1. 用户意图视角
- 用户搜索时使用的属性
- 用户比较商品时的关注点
- 用户购买决策时的考虑因素

### 2. 场景视角
- 使用场景（送礼、自用、办公...）
- 适用环境（室内、室外、户外...）
- 适用时机（节日、季节...）

### 3. 运营选品视角
- 爆款特征属性
- 利润款特征属性
- 引流款特征属性

### 4. 搜索视角
- 搜索高频属性
- 属性可筛性
- 属性可排序性

### 5. 推荐视角
- 推荐强关联属性
- 用户画像属性
- 协同过滤属性

## 分析方法

1. 读取当前属性库
2. 从每个视角分析属性适用性
3. 识别缺失属性
4. 给出补充建议
