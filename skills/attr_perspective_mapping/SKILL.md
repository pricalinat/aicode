# 属性-视角映射分析 Skill

## 目标
从5个业务视角重新组织属性库

## 视角维度

### 1. 用户意图视角
- 用户搜索时使用的属性：品牌、价格、颜色、尺寸...
- 用户比较时的关注点
- 购买决策因素

### 2. 场景视角
- 使用场景：送礼、自用、办公...
- 适用环境：室内、室外、户外...
- 适用时机：节日、季节、日常...

### 3. 运营选品视角
- 爆款属性：高需求、低价格带
- 利润款属性：高毛利、高复购
- 引流款属性：低毛利、高曝光

### 4. 搜索视角
- 筛选项属性
- 排序项属性
- 高频搜索属性

### 5. 推荐视角
- 用户画像属性
- 行为数据属性
- 协同过滤属性

## 分析方法

1. 读取当前属性库 (1004项)
2. 从每个视角分析适用性
3. 标记每个属性的视角标签
4. 识别缺失属性
5. 生成映射报告
