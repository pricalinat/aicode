# Project Analysis Agent Skill - 项目分析优化建议

## 目标
每小时运行一次，分析项目优缺点，提出优化建议。

## 分析范围

### 1. 代码项目分析
- aicode项目
- codex项目

### 2. 书籍写作分析
- book_v2目录

### 3. 工作流程分析
- 对话方式
- 任务执行效率
- 自动化程度

## 分析维度

### 代码分析
- 代码结构
- 性能瓶颈
- 可维护性
- 测试覆盖
- 文档完整度

### 项目进度
- 任务完成度
- 问题解决效率
- 迭代质量

### 工作方式
- 对话风格
- 任务分配
- 自动化程度
- 监控告警
